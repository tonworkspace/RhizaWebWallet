## 🔗 How to Access

### Method 1: From Landing Page
```
Landing Page (/)
    ↓
Scroll to "What Can You Do With $RZC?" section
    ↓
Click "Explore All RZC Utilities" button
    ↓
RZC Utility Hub (/use-rzc)
```

### Method 2: From More Menu
```
Wallet Dashboard
    ↓
Click "More" tab
    ↓
See "RZC Utilities" section (top)
    ↓
Click "Use RZC Everywhere"
    ↓
RZC Utility Hub (/use-rzc)
```

### Method 3: Direct URL
```
Type in browser: /use-rzc
or
Type in browser: /rzc-utility
```

## 🎨 Design Highlights

### Typography:
- Headings: Black weight (900)
- Body: Medium weight (500)
- Labels: Bold uppercase (700)

### Spacing:
- Section padding: 24px (mobile) to 96px (desktop)
- Card padding: 32px
- Gap between cards: 24px

### Borders:
- Card borders: 1px solid
- Rounded corners: 24px (cards), 16px (buttons)
- Hover borders: Primary color with 30% opacity

### Shadows:
- Default: None
- Hover: Large shadow with primary color glow
- Buttons: Medium shadow with primary color
