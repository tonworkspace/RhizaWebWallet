## Key Takeaways

1. **Referral Code = User's Identifier**
   - Each user gets unique code
   - Code is shared via link
   - New users sign up with code in URL

2. **Two-Way Relationship**
   - User B stores "who referred me" (referrer_code)
   - User A stores "who I referred" (via referrer_id in B's record)

3. **Rewards Based on Activity**
   - Not just for signing up
   - Ongoing rewards for referred user's transactions
   - Lifetime earnings potential

4. **Tier System Incentivizes Growth**
   - More referrals = higher commission
   - Encourages users to refer more people
   - Rewards top performers

5. **Database Tracks Everything**
   - Every referral counted
   - Every earning recorded
   - Full audit trail
