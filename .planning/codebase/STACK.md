# Stack

## Environment
- **Runtime**: Node.js
- **Package Manager**: npm
- **Framework**: Vite + React
- **Language**: TypeScript

## Dependencies & Packages
- **Frontend**: React (v18+), React Router DOM (assumed), Tailwind CSS (postcss, autoprefixer).
- **Blockchain**: `@ton/ton`, `@ton/crypto`
- **Backend/DB**: `@supabase/supabase-js`
- **Build tool**: Vite, TS compiled targeting DOM/ES decorators.

## System Tools
- Get-Shit-Done (GSD) CLI for agent workflow management.
- Kiro / Qoder for internal IDE tracking.
