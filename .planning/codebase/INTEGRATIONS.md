# Integrations

## External Systems
1. **TON Blockchain (Mainnet & Testnet)**
   - Utilizes custom smart contracts (Mining, Jettons).
   - API Gateways: TonCenter (`api.toncenter.com`), TonAPI (`tonapi.io`), and Tonhub API.
2. **Supabase (Backend-as-a-Service)**
   - Used for PostgreSQL Database, Authentication (RLS enforced), and edge mechanisms.
3. **Telegram Mini Apps (TMA)**
   - Built to run seamlessly pointing to Telegram bot (`t.me/RhizaCore_bot/launch`).

## Off-Chain Core Data
- Airdrop Tracking & Tasks
- Newsletter management
- Social Media link tracking
- Support ticketing
- SQL DB stored RZC crediting and accounting logic.
