## Color Palette Reference

### Primary Colors (Emerald/Teal/Cyan)
```css
/* Emerald (Primary) */
bg-emerald-50 dark:bg-emerald-500/10
border-emerald-100 dark:border-emerald-500/20
text-emerald-600 dark:text-emerald-400
text-emerald-700 dark:text-emerald-300
text-emerald-900 dark:text-emerald-300

/* Teal (Secondary) */
bg-teal-50 dark:bg-teal-500/10
border-teal-100 dark:border-teal-500/20
text-teal-600 dark:text-teal-400

/* Cyan (Accent) */
bg-cyan-50 dark:bg-cyan-500/10
border-cyan-100 dark:border-cyan-500/20
text-cyan-600 dark:text-cyan-400
```

### Gradients
```css
/* Primary gradient (buttons, progress bars) */
from-emerald-500 to-teal-500
hover:from-emerald-600 hover:to-teal-600

/* Background gradients */
from-emerald-50 to-cyan-50 dark:from-emerald-500/10 dark:to-cyan-500/10
```

### Dark Mode Standards
```css
/* Backgrounds */
dark:bg-[#1a1a1a]        /* Main cards/modals */
dark:bg-[#12141A]        /* Nested elements */

/* Borders */
dark:border-white/10     /* Standard borders */
dark:border-emerald-500/20  /* Colored borders */
```
